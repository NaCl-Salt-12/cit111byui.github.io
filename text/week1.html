What is Data?

Data, Information, Knowledge, Decisions

Data are the bits and pieces, like numbers and characters with no context. We can be given this type of data, but without context it is meaningless to us. We can see, however, the type of data (or datatype) it might be like numbers or characters.

Data in context, are these bits and pieces brought together which give it meaning, and can be understood. They still have the characters and numbers, but now they are wrapped with meaning. But many times even at this point they are not relevant.

Information is a set of data in context with relevance. At least one person can see the relevance in the data. Because information has relevance it is more than just data in context. At a certain place in time, or for a certain person the data makes sense.

Knowledge is information that has been acquired and gives understanding about the importance of the information. Experience with or study of this information increases the awareness or comprehension of the individual. This knowledge can then be used to make decisions.

Everyone will need to make decisions, whether in a career or in every day life. The ability to understand data and, the process in which it becomes knowledge, is critical. There is a certain way data needs to be organized to give it context and eventually become the type of knowledge that allows correct decision making for growth and change.

Decision-making is acting on the knowledge obtained to achieve some benefit. Organizational success depends on these decisions. Systems that these organizations use, must be able to take data, put the data into context, and provide tools for analysis. From this analysis, decisions can be made. A database is designed for just such a purpose.
