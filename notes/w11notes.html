Note: The weekly Teaching Notes only identify week-specific considerations. Please refer to the General Teaching Notes for repeating course facilitation expectations.

Week 11 Outcomes

  • Review concepts including: database design, ERD creation, and Insert commands

Before the Week Begins

  • Students will go over preparation materials before the first class of the week.
  • Consider reviewing the teaching tips and suggestions in the CIT111 - W11 Course Group discussion https://byui.instructure.com/courses/70298/discussion_topics/1457734.

During the Week

  • First of the week:
    • The students should have completed the preparation materials before the beginning of the first class of the week.

    • Instead of a preparation check this week you can briefly demonstrate the additional topics by showing them a few subqueries, how to create an index and how to create a view. They won't use any of these in the project this week or next but you can consider letting them add one of the new topics to their project for extra credit if you'd like.
    • This week students will begin a project that will help the review everything they have learned in previous weeks.
  • Mid-week
    • There will be no quiz this week. This gives them more time to work on their project. 
    • Give them lots of class time this week to complete their projects
  • End of the Week
    • The homework due Saturday will be
      • turning in an ERD that they create, creating insert commands to populate the database and creating a dump file.

Grading/Feedback

  • Instructor: The homework will be instructor or TA graded and will be checking their ERD that they have the database designed properly, checking their SQL file with all their insert commands, and checking the dump file to make sure all the data was inserted correctly.

Looking Ahead

  • Remind students that they must have the first part of this project done correctly in order to get the second half of the project done properly next week. They need to double check everything and get feedback before going onto project part 2 next week.
